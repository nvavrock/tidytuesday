# Papal Encyclicals — TidyTuesday 2026-06-23



Week 25 analysis: papal encyclicals from the Industrial Revolution to the AI Revolution.



## Run



From project root:



```r

source("run_week.R")

run_week("2026_06_23_papal_encyclicals")

```



Or from this folder:



```r

source("../install_packages.R")

source("run.R")

```



## Report



Open [analysis.html](analysis.html) in a browser, open `analysis.qmd` and click **Render** in RStudio, or render from project root:



```r

source("render_week.R")

render_week("2026_06_23_papal_encyclicals")

```



The HTML report embeds static PNGs from `output/` (not plotly). Run `run_week()` before `render_week()` so charts stay in sync.



## Outputs



- `output/` — 8 charts + 4 summary CSVs (300 dpi PNGs for LinkedIn and slides)

- `analysis.qmd` / `analysis.html` — static report (same PNGs as `output/`)

- `NOTES.md` — plain-language briefing for sharing with non-technical readers
