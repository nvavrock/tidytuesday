Takeaway: Noah, Oliver, and Muhammad appear frequently in boys’ top lists; girls’ names show more regional variation. Names that land in multiple regional top-10 lists (e.g. Olivia, Isla) suggest shared cultural trends across the UK. The horizontal axis is the count of babies given each name in that region in 2024 — not a percentage. Each region panel uses its own scale because population sizes differ (England & Wales is much larger than Scotland or NI), so bar length is only comparable within a column, not across regions.

Angle 2: Name uniqueness

Are boys’ or girls’ names more unique? We measure:

  • Distinct names per 1,000 births — higher means more naming diversity
  • Share outside top 100 — higher means fewer parents pick ultra-common names
Code 
plot_uniqueness_over_time(uniqueness) |>
  as_interactive()
Code 
plot_uniqueness_by_sex(uniqueness, interactive = TRUE) |>
  as_interactive()
Code 
summarise_uniqueness(uniqueness) |>
  knitr::kable(digits = 1, caption = "Average uniqueness metrics in the latest available year")
Average uniqueness metrics in the latest available year
Sex avg_distinct_per_1000 avg_pct_outside_top100
Girl 64.0 41.2
Boy 53.7 39.5

Takeaway: Girls’ names are typically more diverse than boys’ names — more distinct names per 1,000 births and a larger share of births using names outside the top 100. This pattern holds across all three regions.
